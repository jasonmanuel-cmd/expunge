---

## Critical Usage Rules

1. Strike Red (#E63946) appears ONLY on primary CTA buttons. Nowhere else.
2. Never stretch, recolor, or add effects to the logo mark.
3. Approved backgrounds: Sovereign Navy, Cloud White, or transparent only.
4. Cleared Green is reserved exclusively for resolved/success states in the UI.
5. No gradients. No drop shadows. Flat surfaces only.
